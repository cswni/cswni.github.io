    Actualizar los cambios en nuestro proyecto local y luego subir a Github

    Abrir la consola en el proyecto

    • Verficar su hemos cambiado algo en el proyecto.

      git status Enter

    • COmo he agregado la clase Persona ahora git detecta que el archivo src/Persona.java no esta siendo observado.

    • Ahora lo añadimos con

      git add * Enter

    • Y confirmamos el cambio con

      git commit -m "Agregando la clase Persona" Enter

    • Finalmente subimos el cambio a Github

      git push origin master Enter
